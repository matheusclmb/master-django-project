Applications:

This project comes with 5 "Applications":

  • Login/Sign up System - Using Django Model and Django Auth, you have to be logged in to enter the site.
  • Weather App - Using OpenWeather API, you can check the weather from cities you choose.
  • Movie Generator - Using TMDB (The Movie Database) API, the site will generate a random movie for you. Limited to 2.000 movies
  • Todo List - Using Django Model, you can create a todo list and delete. With registration date and due date.
  • Contact Form - Using Django Model, you can contact me using the form and django will foward to my email.

Difficulties i had:

While i'm still learning i faced a lot of problems that i managed to fix by searching docs and googling

  • Weather App - When user typed a city that didn't exist, it would return a KeyError

Conclusion
